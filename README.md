## React Project　setup 순서
### `yarn global add npx`
    
### `npx create-react-app jonflix2`
    
### 필요없는 파일 삭제 ( App.js , index.js 만 남기고 삭제 )
    
### git 적용
    
### `.env 파일 생성`
    NODE_PATH=src
    
### `yarn add prop-types`

## 시작
    1. src폴더에 Components폴더 생성
    2. src폴더에 Routes폴더 생성
    3. Routes폴더에 Home, TV, Detail, Search js파일 생성

## `yarn add react-router-dom`
